# Voice Assistant Agent - Backend

This directory contains the Python backend logic and configuration.

## Development

1. Ensure you have [uv](https://github.com/astral-sh/uv) installed.
2. Run `uv sync` to create the virtual environment and install dependencies.
3. Activate the environment with `.venv\Scripts\activate` (Windows) or `source .venv/bin/activate` (Unix).
